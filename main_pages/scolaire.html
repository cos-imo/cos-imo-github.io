Compilateur CanAda

Au cours du module PCL (Projet de Compilation des langages) nous fûmes amenés à étudier la façon dont un compilateur transformait un code source en code éxecutable. Suite à cela, nous avons dû implémenter un compilateur pour CanAda, une fraction du langage Ada.
Note: Je me suis vu contraint de changer de groupe en cours de projet (entre le premier et le second semestre), un nouveau projet PCL sera donc bientôt disponible
